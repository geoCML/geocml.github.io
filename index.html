What makes geoCML so great?

Containerized The geoCML base image uses Docker and Ansible to create and configure your entire GIS deployment. Just pull the base image and run. 
Multi-paradigm Enterprise GIS deployments can be complicated. geoCML makes deployment simple by deploying a desktop and cloud server GIS together.
Lightweight While some enterprise  GIS deployments require 20+ GB of space per machine, geoCML does it all while using less.
Customizable Every GIS deployment is unique. Fork the base image and customize your deployment however you like.
Open Source geoCML is open source software! The geoCML base image packages user friendly, best-in-class open source GIS software including QGIS and PostGIS.
Free Forever The geoCML base image is completely free to use and comes with 0 warranty. The days of expensive licensing fees and user credits are finally over!
